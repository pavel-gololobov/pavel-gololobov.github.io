Описание интерфейса и основных инструментов Eagle

 Пт 27 Август 2010 by admin

Для того, чтобы приступить к изучению редакторов, входящих в состав системы проектирования Eagle, необходимо привыкнуть к интерфейсу и разобраться с основными инструментами программы.

Сразу после запуска, мы можем увидеть главное окно, которое позволяет управлять файлами проектов, библиотеками компонентов и выполнять некоторые другие функции. Данное окно остается открытым на протяжении всего времени работы программы и его закрытие приводит к закрытию всех остальных окон.

[caption id="attachment_261" align="aligncenter" width="553" caption="Главное окно Eagle"][/caption]

В левой части экрана содержится список из различных компонентов системы (по сути это файловый менеджер, отображающий файлы пользователя и служебные файлы структурированно), в правой их описание. На данном этапе я не буду подробно останавливаться на описании всех пунктов из списка, только кратко опишу их назначение.

"Libraries" - список библиотек компонентов и их состав.

"Design Rules" - правила проектирования печатных плат. Определяют возможные зазоры, толщины проводников и т.п. Обычно заполняются исходя из возможностей производства. Позволяют проводить автоматическую проверку.

"User Language Programs" - скрипты, написанные на специализированном языке, позволяющие автоматизировать некоторые операции. К сожалению, я пока не изучал подробно эту возможность.

"CAM Jobs" - создание комплекта документации для производства печатных плат (GERBER-файлы, карта сверловки).

"Projects" - пользовательские проекты. Служит для организации проектов, в том числе содержит несколько примеров, которые рекомендую просмотреть.

Главное меню довольно стандартное, поэтому я не буду подробно останавливаться на его описании.

Рассмотрим основные инструменты редактора электрических схем и печатных плат. Для этого откроем один из примеров, идущих в комплекте программы.

[caption id="attachment_262" align="aligncenter" width="514" caption="Открываем пример"][/caption]

Обратите внимание, что помимо окна с электрической схемой, автоматически открылось окно с печатной платой. Далее, при проектировании своих устройств, эти два окна обязательно должны быть открыты одновременно, т.к. программа поддерживает автоматическую связь между схемой и печатной платой. Внесение изменений в один из этих компонентов сразу же отражается на другом.

[caption id="attachment_263" align="aligncenter" width="507" caption="Редактор электрической схемы"][/caption]

Слева находится панель инструментов, которая содержит как стандартные инструменты, присутствующие во всех редакторах программы, так и специфические для конкретного редактора.

Рассмотрим общие инструменты, которые располагаются вверху, начиная с "Info", который вызывается нажатием кнопки с изображением буквы "i".

Выбор инструмента осуществляется нажатием на соответствующую кнопку и последующим выбором компонента, к которому инструмент будет применяться. Нажмите на кнопку "i", а затем щелкните на любом компоненте в схеме. Появится следующее окно:

[caption id="attachment_264" align="aligncenter" width="304" caption="Окно свойств компонента"][/caption]

В свойствах компонента можно изменить его имя (Name), величину (Value) и некоторые другие параметры. Для разных элементов схемы список свойств будет отличаться. Например, для проводника или шины можно задать имя, стиль, толщину и т.д.

Под кнопкой с изображением цветных квадратиков скрываются настройки отображения слоев схемы.

[caption id="attachment_265" align="aligncenter" width="278" caption="Отображение слоев"][/caption]

Далее, опишу все кнопки по-порядку. В принципе, по рисунку на них не сложно догадаться о функциональном назначении.

  • Перемещение компонента;
  • Клонирование;
  • Отражение;
  • Поворот;
  • Выделение группы компонентов;
  • Внесение изменений;
  • Вырезать;
  • Вставить;
  • Удалить.

При работе с группой компонентов, сначала инструментом"Group" необходимо выделить группу, затем выбрать инструмент, который планируется применить к данной группе и, щелкнув по ней правой кнопкой, запустить процесс.

Сброс любого действия осуществляется нажанием кнопки "Escape".

Для дальнейшего знакомства с интерфейсом и инструментами, рекомендую понажимать все кнопки, которые покажутся вам интересными, попробовать менять настройки, смотреть на реакцию редактора печатной платы (который, напомню, всегда открыт параллельно со схемой). Если не сохранять внесенные изменения, такие эксперименты абсолютно безопасны. Удачи!
