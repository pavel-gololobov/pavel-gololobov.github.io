Исправление кодировки в Zip-архивах

 Вт 30 Октябрь 2012 by admin

Столкнулся с задачей распаковать zip-архив, сделанный в Windows под Linux, русские имена файлов оказались в весьма странной кодировке. Решение нашлось на просторах интернета:

[text]unzip file.zip

convmv -f cp1252 -t cp850 * --notest  && convmv -f cp866 -t utf-8 * --notest[/text]

Создание образа диска и его монтирование в Linux
Пт 05 Март 2010 by admin

Для того, чтобы создать образ компакт-диска, выполните команду:

[shell]dd if=/dev/dvd of=filename.iso[/shell]

Для монтирования образа диска, выполните:

[shell]sudo mkdir /media/image
sudo mount -o loop filename.iso /media/image[/shell]
